# Delegation Management System

Full Stack Project

Tech Stack:

* Node.js
* Express.js
* React.js (Vite)
* MySQL
* JWT Auth
* RBAC
* Recharts

Roles:

* Superadmin
* Admin
* User

Features:

* Authentication
* Delegation CRUD
* Activity logs
* Reports dashboard
* Charts
* Role based access

Setup

1. Install Node
2. Install MySQL
3. Import database/schema.sql
4. Setup .env
5. Run backend
6. Run frontend

Backend

cd backend
npm install
npm start

Frontend

cd frontend
npm install
npm run dev
